# 内容策略与 AdSense 审批

怎么通过 AdSense 审核，怎么最大化广告收入。

## AdSense 审批要求

Google 在批准你之前会审查你的网站。以下是审核标准：

### 必须满足（缺一不可）

| 要求 | 详情 |
|------|------|
| **原创内容** | 不能从别的网站复制。AI 生成的可以，但必须做实质性修改（见下面的"AI 内容审阅指南"） |
| **足够的内容量** | 至少 15 个页面/文章，每篇 300+ 字 |
| **隐私政策** | 说明网站使用了 Cookie、Analytics、广告追踪 |
| **关于页面** | 介绍网站是谁运营的、做什么的 |
| **联系方式** | 邮箱或联系表单 |
| **清晰导航** | 有菜单，用户能找到所有页面 |
| **网站年龄** | 通常至少运行 1-2 周（各地区不同） |
| **自定义域名** | 不能用 Vercel/Netlify 的免费子域名 |
| **HTTPS** | 必须有 SSL 证书（Vercel 自动处理） |
| **ads.txt** | 根目录下的广告验证文件（`fork-site.sh` 自动创建） |

### 常见被拒原因和解决办法

| 被拒原因 | 怎么解决 |
|---------|---------|
| "内容不足" | 多加文章，目标 20+ 篇，每篇 800+ 字 |
| "没有实质内容" | 网站有用的文字太少。每个页面都要有实质性的、有帮助的内容——不能只有一个工具/小部件而没有任何说明文字 |
| "网站建设中" | 删掉所有"即将上线"、"敬请期待"的占位文字和空白板块 |
| "导航问题" | 修复死链，加上清晰的菜单，确保菜单里链接到所有页面 |
| "网站无法访问" | DNS 还没生效，等 1-2 周再申请 |

### 被拒了怎么办？

1. 不要慌，大多数网站第一次都会被拒
2. 认真看 Google 给的拒绝原因（邮件里会写具体原因）
3. 对照上面的表格修改
4. 等至少 **14 天**后重新申请（太快重新申请 = 自动被拒）
5. 期间继续添加高质量内容

### AdSense 审核太慢怎么办？

正常时间：2-14 天。但有时候会长达 4-6 周。

**超过 2 周没消息：**
1. 登录 [https://adsense.google.com/](https://adsense.google.com/) 看页面顶部有没有状态提示或横幅
2. 确认你的站点能正常访问——用浏览器的无痕模式打开试试
3. 去 [Google Search Console](https://search.google.com/search-console) → "网页"里看你的页面有没有被收录（"已编入索引"的数量大于 0）
4. 继续添加内容——内容越多，通过率越高

**收到拒绝邮件：**
1. 仔细看 Google 给的拒绝原因
2. 对照上面的"常见被拒原因"表格修改
3. 等至少 **14 天**再重新申请（太快重新申请 = 自动拒绝）
4. 去 [https://adsense.google.com/](https://adsense.google.com/) 重新申请

> **常见误解**：你不需要每个站点都单独申请 AdSense 账号。一个 AdSense 账号管所有站点。只要有一个站通过审核了，其他站点直接用同一个 `publisher_id` 就行——不需要再次申请。

## 内容创作策略

### Hugo 内容站

写能在 Google 上获得排名、吸引点击的内容：

**高收益领域**（CPC 越高 = 每次点击赚越多）：

| 领域 | 示例话题 | 预估 CPC |
|------|---------|---------|
| 科技 | 教程、工具评测、怎么做 | $0.50-2.00 |
| 金融 | 省钱技巧、投资入门、记账 | $1.00-5.00 |
| 健康 | 健身、营养、养生 | $0.50-3.00 |
| 教育 | 学习方法、在线课程、语言学习 | $0.50-2.00 |
| 家居 | 收纳、简单维修、装饰 | $0.30-1.50 |

> **CPC 是什么？** Cost Per Click（每次点击费用）——访客每点击一次广告你能赚的钱。上面的范围是大概值，实际取决于访客所在国家、具体话题和广告主。来自美国/英国/澳洲的流量，每次点击收入是发展中国家的 5-10 倍。

**写文章的公式：**

```
标题: [数字] + [关键词] + [好处]
例如: "10 个免费在线 PDF 转 Word 工具推荐"

结构:
- 开头（50-100 字）—— 描述问题
- 正文分段 H2（每段 100-200 字）—— 逐个介绍
- 结尾（50-100 字）—— 总结推荐
- 总字数: 800-1500 字
```

### 用 AI 自动生成文章

SiteFactory 支持通过脚本和 GitHub Actions 自动生成文章：

1. 本地运行 `./scripts/generate-content.sh <站点名> <主题>`
2. 或使用 GitHub Actions 的 "Generate Content" 工作流（Actions 标签 → Run workflow）
3. 在 `config.yaml` 中配置 AI 服务（支持 Claude、OpenAI、Gemini、DeepSeek、Moonshot、Zhipu）
4. 定时生成可用 "Scheduled Content Generation" 工作流，每天自动运行

### AI 内容审阅指南

发布任何 AI 生成的文章之前，必须做以下所有检查（每篇大约花 10-15 分钟）：

1. **核实事实**：在 Google 上搜索文章里的关键说法。AI 会编造不存在的统计数据、工具名称、网址或日期。
2. **加入你的声音**：用自己的话重写开头和结尾。每个段落至少加 2-3 句个人观点或经验。
3. **修掉机器味**：AI 喜欢写"综上所述"、"值得注意的是"、"在当今数字化时代"。把这些替换成自然的过渡语。
4. **删除重复内容**：AI 经常在不同段落里用稍微不同的措辞说同一件事。找到并删除冗余部分。
5. **验证链接**：如果 AI 在文章里放了网址链接，逐个点击确认能打开且内容正确。删掉打不开的链接。

> **为什么这么重要？** Google 能识别低质量的 AI 生成内容。直接把 AI 的原始输出发布 = 大概率被 AdSense 拒绝。做了上面这些修改后，内容能通过审核。

### Fork 工具站

工具站不需要博客内容，但需要辅助页面来通过 AdSense 审核：

1. **使用说明**（300+ 字）—— 一步步教怎么用这个工具
2. **关于本工具**（300+ 字）—— 做什么的，给谁用，为什么有用
3. **常见问题**（300+ 字）—— 用户可能问的问题和回答
4. **隐私政策** —— `fork-site.sh` 已自动添加
5. **相关工具** —— 链接到你的其他站点

> **这些页面加在哪？** 在站点根目录（`sites/你的工具/`）下创建单独的 HTML 文件（比如 `how-to-use.html`、`about.html`），然后在主页的导航栏里加上链接指向这些页面。

## SEO 基础

### 页面级 SEO 清单

每个页面都应该有：

```
☐ 标题标签（50-60 字符，关键词放前面）
☐ Meta description（150-160 字符，包含行动号召）
☐ 唯一的 H1 标题
☐ 合理的 H2-H4 层级
☐ 图片都有 alt 文字
☐ 站内链接（链接到自己的其他页面）
☐ 站外链接（链接到权威网站）
☐ 干净的 URL 结构
```

### 技术 SEO

SiteFactory 的 Hugo 模板自动处理以下内容：

- sitemap.xml 自动生成
- robots.txt
- 规范 URL（canonical）
- Open Graph 标签
- JSON-LD 结构化数据
- hreflang 多语言标签

### 免费关键词研究方法

1. **Google 自动补全** —— 在 Google 搜索框里输入话题看自动建议
2. **"大家还在搜"** —— 看 Google 搜索结果页面里出现的"其他人也在问"板块
3. **Google Trends**（[trends.google.com](https://trends.google.com)）—— 对比关键词热度趋势
4. **AnswerThePublic**（[answerthepublic.com](https://answerthepublic.com)）—— 找人们在搜什么问题

## AdSense 优化

### 广告规则

SiteFactory 使用 **Auto-Ads**（自动广告），Google 会自动在最优位置放广告。但要遵守以下规则：

| 规则 | 说明 |
|------|------|
| 每页最多 3 个广告 | 更多广告 ≠ 更多收入 |
| 内容 > 广告 | 广告面积不能超过内容面积 |
| 不能误导用户 | 广告必须看起来像广告 |
| 手机端适配 | 广告不能破坏手机布局 |
| 不能弹窗 | 不能用弹窗遮挡内容 |

### 需要关注的指标

| 指标 | 含义 | 目标值 | 在哪里看 |
|------|------|--------|---------|
| RPM | 每千次浏览收入 | $1-10（因领域而异） | AdSense → 报告 |
| CPC | 每次点击收入 | $0.10-5.00 | AdSense → 报告 |
| CTR | 点击率 | 1-3%（正常范围） | AdSense → 报告 |
| 浏览量 | 总页面浏览次数 | 越多越好 | Google Analytics |

### 危险信号（看到立即排查）

| 信号 | 什么意思 | 怎么办 |
|------|---------|--------|
| CTR > 5% | 点击率异常高——可能有无效点击（机器人、恶意点击、或者有人在反复点你的广告） | 去 Google Analytics 查看流量来源，看有没有可疑的来源。绝对不要点自己的广告。等 Google 自动调查。 |
| CTR 突然变 0% | 广告停止在你的站点上展示 | 去 AdSense 后台看有没有错误提示。确认站点能正常打开。 |
| 收入隔夜下降 > 30% | 可能违反了政策或广告被限制 | 登录 AdSense 看有没有政策警告或"广告投放受限"的横幅。 |
| 出现"受限广告投放"提示 | Google 检测到异常的点击或流量模式，暂时限制了你站点上的广告 | **在哪看到**：登录 AdSense 后台，页面顶部会出现黄色或红色横幅提示"广告投放受到限制"。**怎么办**：(1) 绝对不要点自己的广告 (2) 去 Google Analytics 检查有没有可疑的流量来源 (3) 不用做其他操作——Google 会在 30 天内自动审查并恢复 (4) 不要找 Google 客服，这个是自动处理的 (5) 如果超过 30 天还没恢复，检查所有指向你网站的外部链接，移除可疑的 |

## 月度维护

| 任务 | 频率 | 怎么做 |
|------|------|--------|
| 添加新内容 | 每周 | 新文章或页面 |
| 检查 AdSense 后台 | 每周 | 登录看页面顶部有没有警告横幅 |
| 运行 Lighthouse | 每月 | `./scripts/lighthouse-check.sh --all` |
| 更新监控面板 | 每天（通过 GitHub Actions） | `./scripts/generate-dashboard.sh` |
| 查看流量来源 | 每月 | Google Analytics → 流量获取 → 流量获取概览 |
| 检查死链 | 每月 | 浏览器开发者工具控制台（找 404 错误） |

## 时间线规划

| 周 | 目标 |
|----|------|
| 第 1 周 | 搭建 SiteFactory，创建前 3 个站点 |
| 第 2 周 | 给站点添加内容（每站 15+ 页） |
| 第 3 周 | 提交到 Google Search Console，等待收录 |
| 第 4 周 | 申请 AdSense |
| 第 5-6 周 | 等 AdSense 审核，继续添加内容 |
| 第 7 周起 | 扩大规模：更多 Fork、更多内容、优化收入 |

> **如果超过 6 周还没通过 AdSense：** 不要急——继续添加内容，确保所有页面都在 Search Console 里被收录了（[Search Console](https://search.google.com/search-console) → 网页 → 看"已编入索引"数量）。修改 Google 指出的问题后重新申请。
