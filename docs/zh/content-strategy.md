# 内容策略与 AdSense 审批

怎么通过 AdSense 审核，怎么最大化广告收入。

## AdSense 审批要求

Google 在批准你之前会审查你的网站。以下是审核标准：

### 必须满足（缺一不可）

| 要求 | 详情 |
|------|------|
| **原创内容** | 不能从别的网站复制。AI 生成的内容可以，但需要人工审阅修改 |
| **足够的内容量** | 至少 15 个页面/文章，每篇 300+ 字 |
| **隐私政策** | 说明网站使用了 Cookie、Analytics、广告追踪 |
| **关于页面** | 介绍网站是谁运营的、做什么的 |
| **联系方式** | 邮箱或联系表单 |
| **清晰导航** | 有菜单，用户能找到所有页面 |
| **网站年龄** | 通常至少运行 1-2 周（各地区不同） |
| **自定义域名** | 不能用 Vercel/Netlify 的免费子域名 |
| **HTTPS** | 必须有 SSL 证书（Vercel 自动处理） |
| **ads.txt** | 根目录下的广告验证文件（`fork-site.sh` 自动创建） |

### 常见被拒原因和解决办法

| 被拒原因 | 怎么解决 |
|---------|---------|
| "内容不足" | 多加文章，目标 20+ 篇，每篇 800+ 字 |
| "没有实质内容" | 网站有用的文字太少 |
| "网站建设中" | 删掉所有"即将上线"、"敬请期待"的占位文字 |
| "导航问题" | 修复死链，加上清晰的菜单 |
| "网站无法访问" | DNS 还没生效，过几天再申请 |

### 被拒了怎么办？

1. 不要慌，大多数网站第一次都会被拒
2. 认真看 Google 给的拒绝原因
3. 对照上面的表格修改
4. 等 2 周后重新申请
5. 期间继续添加高质量内容

## 内容创作策略

### Hugo 内容站

写能在 Google 上获得排名、吸引点击的内容：

**高收益领域**（CPC 越高 = 每次点击赚越多）：

| 领域 | 示例话题 | 预估 CPC |
|------|---------|---------|
| 科技 | 教程、工具评测、怎么做 | $0.50-2.00 |
| 金融 | 省钱技巧、投资入门、记账 | $1.00-5.00 |
| 健康 | 健身、营养、养生 | $0.50-3.00 |
| 教育 | 学习方法、在线课程、语言学习 | $0.50-2.00 |
| 家居 | 收纳、简单维修、装饰 | $0.30-1.50 |

**写文章的公式**：

```
标题: [数字] + [关键词] + [好处]
例如: "10 个免费在线 PDF 转 Word 工具推荐"

结构:
- 开头（50-100 字）—— 描述问题
- 正文分段 H2（每段 100-200 字）—— 逐个介绍
- 结尾（50-100 字）—— 总结推荐
- 总字数: 800-1500 字
```

### 用 AI 自动生成文章

SiteFactory 的 n8n 工作流可以通过 Claude API 自动生成文章：

1. 把 `n8n/workflows/content-generation.json` 导入 n8n
2. 配置 Claude API Key
3. 设置定时任务（比如每天运行）
4. 工作流会自动生成 SEO 优化的 Markdown 文章

**重要**：AI 生成的内容发布前必须人工审阅：
- 检查是否有事实错误
- 加入自己的观点和独特见解
- 确保读起来自然流畅
- 关键词不要堆砌

### Fork 工具站

工具站不需要博客内容，但需要辅助页面：

1. **使用说明**（300+ 字）—— 一步步教怎么用
2. **关于本工具**（300+ 字）—— 做什么的，给谁用
3. **常见问题**（300+ 字）—— 用户可能问的问题
4. **隐私政策** —— `fork-site.sh` 已自动添加
5. **相关工具** —— 链接到你的其他站点

## SEO 基础

### 页面级 SEO 清单

每个页面都应该有：

```
☐ 标题标签（50-60 字符，关键词放前面）
☐ Meta description（150-160 字符，包含行动号召）
☐ 唯一的 H1 标题
☐ 合理的 H2-H4 层级
☐ 图片都有 alt 文字
☐ 站内链接（链接到自己的其他页面）
☐ 站外链接（链接到权威网站）
☐ 干净的 URL 结构
```

### 技术 SEO

SiteFactory 的 Hugo 模板自动处理以下内容：

- sitemap.xml 自动生成
- robots.txt
- 规范 URL（canonical）
- Open Graph 标签
- JSON-LD 结构化数据
- hreflang 多语言标签

### 免费关键词研究方法

1. **Google 自动补全** —— 输入话题看建议
2. **"大家还在搜"** —— 看 Google 搜索结果里的相关问题
3. **Google Trends** —— 对比关键词热度
4. **AnswerThePublic** —— 找人们在问什么问题

## AdSense 优化

### 广告规则

SiteFactory 使用 **Auto-Ads**（自动广告），Google 会自动在最优位置放广告。但要遵守以下规则：

| 规则 | 说明 |
|------|------|
| 每页最多 3 个广告 | 更多广告 ≠ 更多收入 |
| 内容 > 广告 | 广告面积不能超过内容面积 |
| 不能误导用户 | 广告必须看起来像广告 |
| 手机端适配 | 广告不能破坏手机布局 |
| 不能弹窗 | 不能用弹窗遮挡内容 |

### 需要关注的指标

| 指标 | 含义 | 目标值 |
|------|------|--------|
| RPM | 每千次浏览收入 | $1-10（因领域而异） |
| CPC | 每次点击收入 | $0.10-5.00 |
| CTR | 点击率 | 1-3%（正常范围） |
| 浏览量 | 总页面浏览次数 | 越多越好 |

### 危险信号（看到立即排查）

| 信号 | 可能的问题 |
|------|-----------|
| CTR > 5% | 无效点击（机器人或恶意点击） |
| CTR 突然变 0% | 广告投放出问题 |
| 收入隔夜下降 > 30% | 可能违反政策或广告限制 |
| 出现"受限广告投放"提示 | Google 在审查你的流量 |

## 月度维护

| 任务 | 频率 | 怎么做 |
|------|------|--------|
| 添加新内容 | 每周 | 新文章或页面 |
| 检查 AdSense 后台 | 每周 | 看有没有警告 |
| 运行 Lighthouse | 每月 | `./scripts/lighthouse-check.sh --all` |
| 更新监控面板 | 每天（通过 n8n 自动化） | `./scripts/generate-dashboard.sh` |
| 查看流量来源 | 每月 | Google Analytics |
| 检查死链 | 每月 | 浏览器开发者工具控制台 |

## 时间线规划

| 周 | 目标 |
|----|------|
| 第 1 周 | 搭建 SiteFactory，创建前 3 个站点 |
| 第 2 周 | 给站点添加内容（每站 15+ 页） |
| 第 3 周 | 提交到 Google Search Console，等待收录 |
| 第 4 周 | 申请 AdSense |
| 第 5-6 周 | 等 AdSense 审核，继续添加内容 |
| 第 7 周起 | 扩大规模：更多 Fork、更多内容、优化收入 |
