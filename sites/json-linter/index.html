    Supported Operations:

    • . - Identity (return as-is)
    • .key - Access property
    • .[] - Array iterator
    • .[0] - Array index
    • .[1:3] - Array slice
    • .[] | .name - Map property from array
    • keys - Get object keys
    • values - Get object values
    • length - Get length
    • select(.age > 18) - Filter by condition
    • map(.name) - Map transformation
    • sort - Sort array
    • sort_by(.name) - Sort by property
    • reverse - Reverse array
    • unique - Unique values
    • first / last - First/last element
    • min / max - Min/max value
